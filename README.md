# Tools

> A library of tools

## Project setup and usage

Install dependencies:

```
yarn install
```

Run development server:

```
yarn run dev
```

Build for produciton:

```
yarn run build
```

Run ESLint:

```
yarn run lint
```

Lint and fix:

```
yarn run lint:fix
```
